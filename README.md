
# TALLER ARBOLES LISTAS


El programa crea un árbol binario en Python usando nodos enlazados. Primero se define la clase Nodo, que guarda el valor y los hijos izquierdo y derecho. Luego se crea la clase Arbol, donde se insertan los nodos y se calculan tres cosas: peso (cantidad de nodos), altura (niveles del árbol) y orden (máximo número de hijos que puede tener un nodo).
Finalmente, el usuario ingresa los nodos por teclado y el programa muestra esos resultados.
